博客诞生记

更新－2016.8.14

折腾无止境，最后选择了Hexo来生成博客，原因是Hexo对主题的支持很方便，还有整个博客的界面、管理也是很给力。
唯一觉得不够方便的是：每次更新博客，整个public文件夹的东西似乎都会引起改动，比较影响上传速度。

我是跟着这里一步步来整起来的。另外，评论、分享等可能需要自定义，比如说这个。

嗯～是的。万能的Github里还有一个叫做Github Pages的衍生项目，我是通过它找到了Jekyll，最后打造出这个博客的。免费、无限流量、简单易用是它的优点，所以，今后有地儿写(fa)字(lao)了(sao)。

关于Github Pages，官网有一句话概括很到位的话：

GitHub Pages 基于 Jekyll 构建，你可以轻而易举地在 GitHub 上免费发布网站——自定义域名等等。

嗯～直接贴链接实在点：

  • Github Pages基础教程
  • Jekyll介绍
  • 使用Jekyll建立Github Pages个人网站

ps1：关于Jekyll的一个配置文件Gemfile，建议source配置改为 ‘https://ruby.taobao.org/‘ 在国内会合适些。

ps2：关于Web开发相关的一些基础，我查了一些挺有帮助的资料，比如说这个mozilla MDN－了解Web开发－Web开发入门
