关于常见的协议－归/解档、深/浅拷贝

NSCoding

很多时候，我们需要将我们的处于内存里的对象保存到磁盘里，方便下次使用这个对象。其中，这个保存过程叫做归档－archive。相反，把磁盘里的数据变成对象，则称为解档－unarchive。

而unarchive中包含把二进制数据转化成对象的过程，称为序列化；相反，把一个对象变成磁盘里的文件这个归档过程中，对象转变成二进制数据这个过程，就就反序列化。

或者这样理解：序列化是把看似凌乱的二进制数据变成有序的结构体（内存里的对象其实就是以结构体的形式存在的），反序列化就是把有序的结构体对象变成看似凌乱的二进制数据。

NSMutableCopying

此协议仅声明了一个方法：mutableCopyWithZone: ，但是可变的拷贝常常是通过 mutableCopy 调起。这个mutableCopy 方法是在所有的 NSObject 类中定义，且只是简单的调用 mutableCopyWithZone: 而已。

如NSCopying协议一样，如果父类实现了NSMutableCopying协议，那么在实现子类的mutableCopyWithZone: 方法时，需先调用父类的 mutableCopyWithZone: 。

关于Git与Tower的一些使用备忘

Git是目前最流行的版本管理工具，Git的功能相当强大，以git为核心的可视化工具有免费的Github的 Github Desktop、SourceTree以及收费Tower等。嗯我个人对比使用过之后，觉得还是Tower比较符合我自己。本文将会介绍下我在进行项目版本管理的时候比较重要的一些操作存做备忘。

本地回滚到某一个版本。

History，选中你希望回滚的版本，右键，Reset Head to “xxxx”。

说明：若要保留旧版本到当前版本的修改内容，记得勾上Keep Changes。否则修改内容应该是会被丢弃的。

关于文件忽略，根据使用情境会有所不同。

  • 文件/文件夹之前已经被添加进Git中管理了，现在想把它们忽略掉（这些文件的变化git不去做管理）。

    你可能需要动用git命令: git rm -r --cache _site/ 递归的把仓库根目录下的_site目录及其子目录的所有内容剔除git的管理，也就是以后这个文件还会在本地，但是服务器上的这会被删除。注意：除此之外，还需要把 _site/目录添加到 .gitignore 配置文件中，这样关于 _site目录的所有修改，git都不会监控。通常这种做法的目的是，不希望本地的数据被提交上到服务器上。

  • 其它一般的情况，直接配置.gitignore文件一般都能解决。

纠正上一次的提交日志。

Workingspace － Working Copy － 勾上 Amend Commit xxx选项，如果本次暂存区没有东西要提交，那么就是直接修改上次的提交日志。如果有，那么就是合并此次以及上次的提交，日志本次为准。

合并多个commit。

方法1：回退版本，保留变更内容，提交新版本、日志。此法简单粗暴，有些情况不大适合使用。

​
方法2：概括起来就是，r－重新编辑日志。f－向前合并commit，丢弃本次commit的日志。p－保留commit与日志，一般情况都能解决commit的合并需求。

  • 前提

    你的commits都还没有被push出去，你想根据功能合并多个commit。

7月杂谈

感受

  1. 谈谈幸福感

    现如今的中国，普通民众的幸福指数相对较低，而直接影响这个指数的是：我们需要工作多长时间才能换来属于自己的车子、房子。当前的中国社会，一个普通大学生毕业后到攒足自己房子首付的钱需要多少年，我们可以大体算一算。北上广深，其中的广州是房价最低的城市，郊区黄埔、萝岗、番禺、南沙等地的楼盘均价算1.5w一方，80平的小三房，其它凌杂的不算了，就算120w。三成首付36w。好，就拿这36w说说。普通的家庭，36w可是天文数字，如果你毕业后的目标就是这笔钱，那么三五年内，你的日子将会过的很不像样。再说下去就想哭了。
    幸福感真的低得快到马路边上了。。。

    不过，坦白来说，两个人一起生活、一起为我们的房子、车子、甚至是以后的孩子奋斗，相互体谅，可能这就是再平凡不过的幸福感了吧。。。

  2. 关于结婚

    如今毕业两年，业已到了成家的年纪（其实主要是女方再不结婚就已经严重晚婚晚育了）。关于结婚相关事宜比如求婚、婚纱照、酒席、婚礼等的花销，我简单算了一笔，得上到10w（当然，份子钱我没算进去）。嗯，先不说房子车子什么的，就连结个婚，感觉都很辛苦。但是，想想结婚是人生第一件大事，就算辛苦，咬咬牙也就过去了。

一些事

  1. 身边早我毕业两三年的同事，已经付了40w的首付。想想自己，哎。。。还是不想了，还是好好敲代码，练好手艺吧。

  2. 一哥们今天去下订思域10。不得不说的是，本田对于自家员工的购车补贴真是令人羡慕，每月1500的油费封顶补贴、1000的月供补助。嗯，关于买车的需求，还是应该好好衡量下了。但肯定是等到年后再考虑了，毕竟一个月3k的车贷，油费和停车费1.5k，加起来4.5k可不是小数目，尤其是现在差不多每月3.2k的房租水电的情况下，是万万不能想象的。。。

  3. 七月初，女票急性肠胃炎；七月末，急性寻麻疹。这都什么跟什么，累死我了。。。
